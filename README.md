### 🏠 [Homepage](https://github.com/GloomyGhost-MosquitoSeal/ColorCout)

### ✨ Demo
![Demo](https://user-images.githubusercontent.com/12003087/78259083-c6a6ff80-752e-11ea-8d16-0a45fcc811c9.png)

## Usage

```sh
std::cout << cc::red << " red "
          << cc::yellow << " yellow "
          << cc::green << " green "
          << cc::reset << " reset"
          << std::endl;
```
